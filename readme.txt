== Description ==
<h4>Please Note: This Plugin Requires API Key</h4>

This Plugin will install the Listing Search template pages: 

1. Search -- Display the search
2. Search Results -- display the list of search results
3. Listing Details  -- display the details page
4. Active Listings - display the list of active lisings of the specified agent id on the plugin setting
5. Sold Listings - display the list of solds of the specified agent id on the plugin setting 


== Installation ==

1. Upload the plugin folder to "/wp-content/plugins/" directory.
1. Go to: WP Admin >> Plugins >> Activate plugin
1. Go to: WP Admin >> AgentEasy >> Set the API Key.
1. Done.
